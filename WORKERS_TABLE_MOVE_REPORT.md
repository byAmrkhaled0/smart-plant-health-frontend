# Workers Table Move Fix

تم تنفيذ التعديل المطلوب:

- تم حذف جدول العاملين/الصلاحيات من صفحة لوحة الإدارة `/admin`.
- تم نقل جدول العاملين إلى صفحة العاملون الأساسية `/workers`.
- جدول صفحة العاملون يعرض الآن: اسم العامل، الدور، القطاع، عرض، إضافة، تعديل، حذف.
- صفحة لوحة الإدارة بقيت للإدارة العامة فقط: بيانات المزرعة، إدارة القطاعات، الأجهزة، التنبيهات، التقارير.
- لم يتم تغيير هوية الموقع أو ألوانه أو الصفحة الرئيسية.

ملاحظة: لم يتم تشغيل `npm run build` داخل البيئة هنا بسبب عدم اكتمال تثبيت `node_modules`، ويجب تشغيله محليًا بعد فك الضغط.
