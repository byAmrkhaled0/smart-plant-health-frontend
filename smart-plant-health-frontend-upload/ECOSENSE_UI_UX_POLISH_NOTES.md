## Main Changes

### General UI/UX
- Reduced visual noise across the website.
- Softened the color palette using calmer green and neutral tones.
- Unified card, panel, button, badge, and shadow styles.
- Increased spacing between sections and cards.
- Improved typography, line-height, headings, labels, and text hierarchy.
- Reduced heavy gradients and strong shadows to make the website calmer for presentation.

### Dashboard
- Kept the main dashboard focused on KPIs, latest readings, and the main action.
- Moved secondary content into a collapsible details section to reduce clutter.
- Improved KPI sizing and spacing.

### Farm Management
- Reworked service cards to be smaller, clearer, and less crowded.
- Improved device cards, task cards, alerts, and farm report layouts.
- Improved labels and field spacing so content does not look merged.

### Workers
- Improved workers layout and generated account card readability.
- Organized worker details into clean lines with labels.
- Improved mobile spacing and button layout.

### Reports
- Reduced report preview visual weight.
- Improved report cards with clearer spacing, labels, and short summaries.
- Kept Export PDF and CSV functionality unchanged.

### Alerts
- Improved alert card readability.
- Added cleaner separation between alert type, reason, sector, time, and severity.
- Improved alert actions on mobile.

### Settings
- Cleaned card spacing and profile card sizing.
- Kept the simple Appearance toggle style as requested.
- Preserved language switching and RTL/LTR behavior.

### Mobile
- Added stronger mobile-first overrides.
- Prevented cards, buttons, and grids from overlapping on small screens.
- Converted complex grids to one-column layouts where needed.
- Improved button tap sizes and spacing.
- Hid non-essential dashboard workflow strip on mobile.
