# Ecosense AI Frontend

Original Ecosense AI / Smart Plant Health frontend restored.

## Run locally

```bash
npm config set registry https://registry.npmjs.org/
npm install --legacy-peer-deps
npm run dev
```

## Notes

- This is the Ecosense frontend only.
- The favicon uses the Ecosense project logo.
- The app keeps the original Ecosense interface and routes.
- Backend URLs have safe defaults inside the code and can still be overridden from deployment environment variables if needed.
